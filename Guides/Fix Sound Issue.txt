Install & Configure Sound (Source: https://mysudo.wordpress.com/2015/03/27/unmute-pulseaudio-sound-in-xfce-debian-jessie/)

--Unmute
	1) alsamixer *Press M to unmute/mute*
	
--Show Components (su -)
	1) xfconf-query -lc xfce4-mixer
	
--Install Missing Package (gstreamer0.10-pulseaudio)
	1) sudo apt-get install gstreamer0.10-pulseaudio
	
--Check Commands
	1) xfconf-query -c xfce4-mixer -p /sound-card
	2) xfconf-query -c xfce4-mixer -p /sound-card -s PlaybackInternesAudioAnalogStereoPulseAudioMixer
